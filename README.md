# Lyryx Backend

Данный репозиторий содержит Backend-часть дипломного проекта Lyryx.

## Стек технологий

- Go 1.24+
- PostgreSQL 15
- MinIO
- Ollama
- Docker & Docker Compose

## Инструкция по запуску

### 1. Настройка окружения

Убедитесь, что у вас установлены Docker, Docker Compose, Go и утилита make.
Создайте файл `.env` в корне проекта и укажите необходимые переменные для базы данных, MinIO и Ollama.

### 2. Запуск инфраструктуры

Поднимите необходимые сервисы (БД, MinIO, Ollama) в Docker:

```bash
make app-up
```

### 3. Применение миграций и тестовых данных

Накатите миграции на базу данных и загрузите тестовые данные:

```bash
make migrate
```

### 4. Настройка Ollama

Загрузите модель для Ollama (по умолчанию используется модель, указанная в переменной `OLLAMA_MODEL_NAME`):

```bash
make setup-ai
```

### 5. Запуск сервера

Запустите основное приложение:

```bash
go run cmd/app/main.go
```

## Документация API

Для генерации актуальной Swagger документации используйте команду:

```bash
make docs
```
